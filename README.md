## 기능 설명

### [계산 기능]

- 사칙연산 및 %, 제곱, 제곱근, 음수/양수 변환 등의 기능을 가진 계산기입니다.

```
- Vanilla JS 로 class를 만들어 기능들을 구현하였습니다.
- 숫자들을 입력 후 수식을 누르면 위에 회색글씨로 계산 과정이 나옵니다.
```
